# 学生网上选课管理系统 java99

## 项目概述

本项目为学生网上选课管理系统，采用Java EE技术栈开发，实现前后端分离。系统基于Springboot和Vue.js构建，遵循MVC三层架构，确保了系统的可维护性与扩展性。

## 技术栈

- **后端：** Spring + SpringBoot + Mybatis
- **前端：** Vue.js
- **构建工具：** Maven
- **数据库：** MySQL

## 功能模块

- 学生信息管理
- 课程信息管理
- 选课功能
- 课程表查询
- 成绩管理

## 系统角色

- 学生
- 教师
- 管理员

## 运行环境

- **操作系统：** Windows/Linux/MacOS
- **Java版本：** JDK 1.8+
- **数据库：** MySQL 5.7+
- **Web服务器：** Tomcat 8.5+

## 部署步骤

1. 环境准备：确保JDK和MySQL已正确安装
2. 数据库初始化：导入MySQL数据库脚本
3. 后端部署：编译后端代码，部署到Tomcat服务器
4. 前端部署：编译前端代码，部署到Web服务器
